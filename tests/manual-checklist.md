# Manual acceptance checklist

- [ ] Launch the generated simulator directly from `file://` and confirm it makes zero network requests.
- [ ] Confirm the default room geometry.
- [ ] Exercise all five source types.
- [ ] Edit the room and confirm invalid topology is rejected.
- [ ] Exercise undo and redo.
- [ ] Confirm off-room sources are removed.
- [ ] Rotate a source by 45°.
- [ ] Adjust source gain, polarity, and delay.
- [ ] Exercise all three analysis views.
- [ ] Inspect the listening-point response.
- [ ] Exercise ceiling modes.
- [ ] Exercise worker execution, fallback execution, and cancellation.
- [ ] Save and load presets and JSON.
- [ ] Navigate by keyboard and inspect the screen-reader object list.
- [ ] Confirm reduced-motion behavior.
- [ ] Inspect color and contour legends.
- [ ] Complete numerical validation.
- [ ] Measure preview, solve, and cancellation timing.
